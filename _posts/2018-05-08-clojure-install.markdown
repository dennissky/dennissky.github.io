# Clojure环境配置  

####基于mac的环境配置

1. 利用brew安装clojure  
    brew是mac下必备的软件,具体安装方法参照 <https://brew.sh/index_zh-cn.html>,安装好brew我们去clojure官网,
    <https://www.clojure.org/guides/getting_started> 一条命令就可以成功的安装好clojure的环境  
    
2. 配置idea开发环境  
    idea插件搜索Cursive,安装好后重启idea。
